# CC - 写作社区微信小程序

一个微信小程序版的写作/文章社区平台，支持用户登录、文章发布、浏览、评论、点赞、关注、举报、审核等功能。

## 功能特性

- 微信登录与用户管理
- 文章发布（支持文字+图片混合排版）
- 文章按专题/标签分类浏览
- 评论系统（好评/差评）
- 文章审核机制
- 用户关注/粉丝系统
- 图片上传

## 技术栈

- 微信小程序原生开发（WXML / WXSS / JS）

## 运行方式

1. 使用微信开发者工具导入项目
2. 配置 `project.config.json` 中的 AppID
3. 编译运行
